Data e Hora P1

Na aula de hoje aprendemos sobre date e hora em JS, para isso utilizamos a interface Date que nos permite ter acesso a data, minutagem e fuso-horário. Para utilizarmos isso, criamos um objeto para instanciar essa interface, e por padrão vai retornar a data e hora do SO. Essa variável agora vai possuir 4 categorias de métodos: getData/Hora/Fuso, getUTCData/Hora/Fuso, setData/Hora/Fuso e setUTCData/Hora/Fuso. Basicamente, get vai retornar algum valor em específico do objeto instanciado, set será usado para alterar um valor, e o UTC está relacionado ao fuso-horário do usuário.

Estrutura

        "use strict"
        const data = new Date() //Podemos passar valores dentro os parênteses(construtor do objeto)
        data.getDate() //Dia do mês
        data.setFullYear() //Altera o ano
        data.getUTCHour() //Hora no fuso-horário

Nesse exemplo instanciamos um objeto da interface Date() e mostramos como usar os métodos vistos acima.

Vamos analisar o código do projeto:

        let data = new Date()
        data.setFullYear(2024)
        console.log(data)
        console.log("Dia do mês: "+data.getDate())
        console.log(`Dia da semana: ${data.getDay()+1}`)
        console.log("Ano: "+data.getFullYear())
        console.log("Hora: "+data.getHours())
        console.log("Milisegundos: "+data.getMilliseconds())
        console.log("Minutos: "+data.getMinutes())
        console.log(`Mês: ${data.getMonth()+1}`)
        console.log("Segundos: "+data.getSeconds())
        console.log("Timestamp: "+data.getTime())
        console.log("Fuso-horário: -"+data.getTimezoneOffset())
        const divData = document.querySelector("#data")
        divData.innerHTML=`${data.getDate()<10?`0${data.getDate()}`:data.getDate()}/${data.getMonth()<10?`0${data.getMonth()+1}`:data.getMonth()}/${data.getFullYear()}`

No código acima declaramos um objeto da classe Date(), logo abaixo setamos o seu ano para 2024. Também temos uma séria de consoles.log() que exibem os dados do objeto por completo e individualmente, no mês e dia da semana é necessário incrementar em 1, pois eles funcionam a base de um array, retornando a posição do array, ou seja, o mês 1 está na posição 0, retornando 0, após incrementar vai retornar 1. A mesma coisa se aplica para o dia da semana. Por fim declaramos uma div, onde alteramos seu innerHTML para que ela receba a data como utilizamos aqui no Brasil, entre barras. Utilizamos os operadores ternários para exibir o dia e mês sempre com 2 digitos, onde caso o mês ou dia possuam apenas 1 digito o 0 será adicionado.
